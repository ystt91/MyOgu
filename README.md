<h1>반려동물 커뮤니티 5959</h1>
<div>반려 가족 1000만 시대를 맞이하여 또 하나의 가족인 반려동물과 함께하는 집사들을 위한 커뮤니티를 만들었습니다.</div>
<div>AWS의 RDS를 활용하여 데이터 베이스를 구축하고 EC2 서비스를 활용하여 배포 했습니다.</div>
